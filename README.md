# 数学读题小助手

逐步引导做题的前端网页工具，适合学生按步骤完成几何读题练习。

## 在线地址

- **学生入口**：[https://louise427-ux.github.io/math-helper/](https://louise427-ux.github.io/math-helper/)
- **教师预览**：[https://louise427-ux.github.io/math-helper/preview.html](https://louise427-ux.github.io/math-helper/preview.html)

学生入口需要先填写班级和姓名；教师预览会直接进入题目列表，不会记录学生信息。

## 本地开发

```bash
npm run validate
python3 -m http.server 8080
```

浏览器打开 `http://localhost:8080/index.html`。
